## Network

### packet의 시작점 부터 도착점까지의 상세한 과정 설명하기 위한 지식 필요.

* ARP 설명
* IP만을 통해서 Destination를 찾는 법 ( 네트워크 안에 Destination 정보가 없으면 상위 네트워크에서 처리하도록 전달 )
* 같은 IP를 동시에 할당 받으면 생기는 문제(IP를 Static으로 사용)
* Port 설명
* TCP-IP 연결해서 설명
* 3 way-handshake 상세설명
